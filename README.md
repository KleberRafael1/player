# 🎬 Alura Player

Projeto desenvolvido durante os estudos na **Alura**, com foco em **Flexbox e responsividade no CSS**.

## 🌐 Deploy

Acesse o projeto online:

https://player-gules.vercel.app

## 💻 Sobre o projeto

O **Alura Player** é uma interface inspirada em plataformas de streaming de vídeo, desenvolvida para praticar a construção de layouts modernos utilizando **Flexbox**.

O projeto foi criado com o objetivo de entender melhor a organização de elementos na página e como adaptar o layout para diferentes tamanhos de tela.

## 🎯 Funcionalidades

- Interface de player de vídeos
- Layout utilizando **Flexbox**
- Design responsivo
- Estrutura organizada de componentes

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- Flexbox
- Vercel (Deploy)

## 📱 Responsividade

O projeto foi desenvolvido considerando diferentes dispositivos:

- Desktop
- Tablet
- Smartphone

## 🧠 Aprendizados

Durante o desenvolvimento foram praticados:

- Layout com **Flexbox**
- Organização de elementos na página
- Responsividade com CSS
- Estruturação de interfaces web
